# Next.js Portfolio 

A modern, responsive portfolio website built with Next.js, Tailwind CSS, and Aceternity UI components.

## Tech Stack

- Next.js
- Tailwind CSS
- Aceternity UI components
- Three.js (for 3D/Globe components)
- Framer Motion (for animations)
